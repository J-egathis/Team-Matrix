**TeamMatrix** is a modern Employee Work Management System designed to simplify task management, employee tracking, attendance monitoring, and organizational workflow. It provides a secure role-based platform where Main Admins, Admins, and Employees collaborate efficiently through an intuitive dashboard.

Built with a modern 2026 UI/UX, TeamMatrix helps organizations improve productivity by managing daily tasks, monitoring employee performance, recording attendance, and generating insightful reports.

# рџљЂ Installation

### Clone Repository

```bash
git clone https://github.com/J-egathis/Team-Matrix.git
```

### Navigate

```bash
cd Team-Matrix
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

### Start Server

```bash
npm start
```

Development Mode

```bash
npm run dev
```
